RDialogs

A ruby wrapper for ncurses dialog and newt whiptail.

RDialogs allows ruby scripts to display dialog boxes to the user for informational purposes, or to get input from the user in a friendly way.

Installation

Add this line to your application's Gemfile:

gem 'rdialogs'

And then execute:

$ bundle

Or install it yourself as:

$ gem install rdialogs
